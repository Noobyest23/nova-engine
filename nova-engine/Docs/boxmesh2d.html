BoxMesh2D

Inherits: Mesh2D > Asset

Description

BoxMesh2D is a primitive geometric asset that generates a standard rectangular quad. By default, this mesh is created as a Unit Box (1.0 x 1.0) centered at the origin.

Scene Usage

Box meshes can be defined in scene files and assigned to renderers. They are often used as the default geometry for Sprites.

[BoxMesh2D]
tag = "standard_quad"
[/BoxMesh2D]

[Sprite2D]
mesh = "standard_quad"
image = "my_texture"
[/Sprite2D]

C++ Usage

// Create a new box mesh
BoxMesh2D* box = new BoxMesh2D();

// A BoxMesh2D contains 4 vertices and 6 indices (2 triangles)
uint32_t indices = box->GetIndexCount(); // Returns 6

Vertex Specifications

The vertices are generated with the following default attributes:

  • Positions: Four points ranging from -1 to 1 on both axes.
  • UV Coordinates: Standard 0-1 mapping (Bottom-Left: 0,0 | Top-Right: 1,1).
Note: To change the dimensions of a BoxMesh2D, it is recommended to adjust the Scale property of the object it is attached to, rather than modifying the vertex data directly.
