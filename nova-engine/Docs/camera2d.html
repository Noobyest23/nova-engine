Camera2D Class

Inherits: Object2D > Object

Description

The Camera2D object determines which portion of the 2D world is rendered to the screen. It uses an orthographic projection where the camera's position represents the center of the view.

Key Features

  • Auto-Resizing: Automatically listens to window resize events to maintain correct aspect ratios.
  • Zooming: Adjusts the visible world area without changing the window size.
  • Centric Workflow: Position (0,0) is the middle of the screen.

Public Methods

Method Return Type Description
SetZoom(float)voidSets the magnification level. Values > 1.0 zoom in; values < 1.0 zoom out. Min value is 0.001.
GetZoom()floatReturns current zoom level.
activeboolProperty. If true, the renderer uses this camera for the current frame.

Scene Usage

[Camera2D]
tag = "MainCamera"
position = Vector2(0, 0)
zoom = 1.5
active = true
[/Camera2D]

NovaScript Interface

func Update(dt) {
    # Simple camera follow logic
    var player = this.GetParent().FindChild("Player")
    if player {
        this.SetPosition(player.GetPosition())
    }

    # Zoom in/out with controls
    if Input.IsKeyPressed(KEY_E) {
        this.SetZoom(this.GetZoom() + dt)
    }
}
Note: While multiple cameras can exist in a scene, only the one marked active = true will be used by the rendering pipeline. If multiple are active, the behavior depends on their order in the scene tree.
