Object

Inherits: None | Inherited by: Object2D

Description

The Object class is the fundamental building block of the Nova Engine scene system. It represents a generic node in the scene hierarchy, supporting parent-child relationships, hierarchical updates, and extensibility. Each Object can have a name, visibility and pause state, an attached script, and any number of child objects. Core engine events such as Ready, Update, Input, and Draw are propagated through the object tree, allowing for modular and reusable scene structures. The Object class provides methods for managing children, searching by name or path, and safely deleting objects. It serves as the base class for all scene elements, and can be extended to create custom behaviors or specialized node types.

In NovaScript all properties from an object are wrapped into one variable on the object: this. Properties are added after the script is put onto the object meaning that this is inaccessable before Ready()

AddChild(my_object: Object)

  • Parameters: my_object (Object): The object to add as a child.
  • Returns: null

Adds an object to this's children if my_object doesn't already have a parent. If two children have the same name then my_object will be renamed to "my_object_name-1".

include "engine-ObjectDB" as db

var sprite = db.Sprite2D()
this.AddChild(sprite)

FindChild(name)

  • Parameters: name (string): The name of the child to find.
  • Returns: Object or null

Returns the first occurrence of an Object among this object's children whose name matches name, or null if no child has a matching name.

var child = this.FindChild("Sprite2D")
if (child) {
    // Found child
}

GetChild(index)

  • Parameters: index (int): The index of the child to retrieve.
  • Returns: Object or null

Returns the child Object at the specified index. If index is out of bounds, returns null.

// returns the first child of this object 
var child = parent->GetChild(0)

GetChild(path)

  • Parameters: path (string): The path of the child to retrieve.
  • Returns: Object or null

Returns the child Object at the specified index. If index is out of bounds, returns null.

var child = parent->GetChild("child_object/child_child_object")

GetChildren()

  • Parameters: None
  • Returns: Array[Object]

Returns an array containing all of this object's children.

include "string" as str
var children = this.GetChildren()

var i = 0
for child : children {
    // Changes every childs name to their index in the children list
    child.name = str.ToString(i)
    i += 1
}

GetParent()

  • Parameters: None
  • Returns: Object or null

Returns this object's parent Object, or null if it does not have a parent. 

var parent = this.GetParent()

Delete()

  • Parameters: None
  • Returns: null

Deletes this Object, after Delete is called code can still be executed for the rest of the current frame. In order to clean up after calling the function, the object is deleted on the next frame

if (!this.visible) {
    // deletes this object if it is not visible
    this.Delete()
}

name

  • Type string

The current name of the object

visible

  • Type bool

If visible is false then this and all of its children will not be drawn

paused

  • Type bool

If paused is false then this and all of its children will not recieve Ready, Update, and Input calls
