Object2D

Inherits: Object | Inherited by: Camera2D, VisualInstance2D

Description

The Object2D class is the base class for all 2D entities within the Nova Engine. It extends the base Object class by adding a 2D transform system, including position, rotation, and scale. It supports a hierarchical transform inheritance; when a parent Object2D moves, rotates, or scales, its children follow suit. It utilizes a "dirty flag" caching system (_dirty_local and _dirty_global) to optimize matrix calculations, only recalculating transforms when a property has actually changed.

GetPosition() / SetPosition(pos: Vector2)

  • Type: Vector2

Gets or sets the position of the object relative to its parent. If the object has no parent, this is equivalent to its global position.

this.SetPosition(Vector2(100, 200))
var pos = this.GetPosition()

GetRotation() / SetRotation(radians: float)

  • Type: float (Radians)

Gets or sets the local rotation of the object in radians. The engine automatically wraps values using TAU (2/pi) to keep the rotation within a 0 to 2/pi range.

// Rotate 45 degrees
this.SetRotation(0.7853)

GetScale() / SetScale(scale: Vector2)

  • Type: Vector2

Gets or sets the local scale of the object. A scale of Vector2(1, 1) is the default size.

this.SetScale(Vector2(2, 2)) // Double the size

GetGlobalPosition() / SetGlobalPosition(pos: Vector2)

  • Type: Vector2

Sets the object's position in world space. If the object is a child, the engine calculates the inverse transform of the parent to determine the correct local position.

// Moves the object to the center of the screen regardless of where its parent is
this.SetGlobalPosition(Vector2(960, 540))

GetGlobalRotation() / SetGlobalRotation(radians: float)

  • Type: float

Returns the total rotation of the object, including the rotation inherited from its parent hierarchy.

GetGlobalScale() / SetGlobalScale(scale: Vector2)

  • Type: Vector2

Returns the final scale of the object after multiplying its local scale with all parent scales in the hierarchy.
