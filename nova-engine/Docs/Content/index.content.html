Welcome to the Documentation of Nova Engine

Nova Engine is a completely free game engine made by a solo developer—powered by caffeine, late nights, and too much free time.

What is Nova Engine?

Nova Engine is an in-development game engine. It uses a class-based system for scene objects. As of right now, I would not recommend trying to actually make a game with Nova Engine. It's still very early in development and is mostly just fun to play around with.

The Documentation (just like the engine itself) is also still in development. There may be pages missing for some things. Make sure to write a bug report for any pages that are missing.

Get Involved
