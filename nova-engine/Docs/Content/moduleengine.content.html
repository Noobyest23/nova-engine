Engine Module

The engine module provides core system functionality, allowing scripts to interact with the application window, manage project global settings, and control the engine's execution flow.

include "engine" as engine

API Reference

Type / Return Name
Window GetWindow()
null SetProjectPath(path)
null Exit(...)
string start_scene
bool supress_warning_popup
bool supress_error_popup

Window Object

The GetWindow() method returns a reference to the application window. This object exposes the following methods:

  • GetName() -> string: Returns the window title.
  • SetName(name: string): Sets the window title.
  • GetSize() -> Vector2: Returns the window dimensions in pixels.
  • SetSize(size: Vector2): Resizes the window.
var win = engine.GetWindow()
win.SetName("Nova Engine - Project Alpha")
var size = win.GetSize()

SetProjectPath(path: string)

Sets the base directory for asset loading and project resources. If the path does not end with a slash, the engine automatically appends one. Primarily used in the Init script

Exit(...)

Immediately stops engine execution and optionally pushes an error message.

if (player == null) {
    engine.Exit("Player object failed to initialize!")
}

Module Variables

These variables are directly accessible on the engine module instance and represent the current engine state:

  • start_scene: A string path to the initial scene file (.nova) used when the engine starts.
  • supress_warning_popup: A boolean toggle to prevent warning dialogs from interrupting execution.
  • supress_error_popup: A boolean toggle to prevent error dialogs from interrupting execution.
