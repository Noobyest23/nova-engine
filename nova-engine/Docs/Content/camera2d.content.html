Camera2D

Inherits: Object2D | Inherited by: None

Description

The Camera2D class is a specialized 2D node used to control what part of the scene is rendered to the screen. It automatically handles window resizing and provides an orthogonal projection centered on the camera's position.

In the Nova Engine, the Camera2D works by transforming the view matrix based on its position and rotation. It supports zooming, where a higher zoom value makes objects appear larger (closer), and a lower value reveals more of the scene.

Properties

Type Name
float GetZoom() / SetZoom(float: zoom)
bool active

GetZoom() / SetZoom(zoom: float)

  • Type: float

Sets the zoom level for the camera. A value of 1.0 represents a 1:1 scale with the window resolution. Values below 0.001 are automatically clamped.

this.SetZoom(0.1)
var zoom = this.SetZoom()

active

  • Type: bool

Determines if this camera is currently being used for rendering. If multiple cameras exist in the scene, the engine uses the first active one it finds in the tree.

Internal Behavior

The Camera2D uses a 2D Orthographic projection with the following bounds:

// Projection Calculation
float hw = (viewportWidth / 2.0f) / zoom;
float hh = (viewportHeight / 2.0f) / zoom;
proj = glm::ortho(-hw, hw, -hh, hh, -10.0f, 10.0f);

This ensures that the point (0, 0) is always the center of the screen regardless of the window aspect ratio.
