Classes and their Loadable Values

Assets

  • FileImage: Used for loading images from files
  • BoxMesh2D: A mesh with a primitive box shape
  • CircleMesh2D: A mesh with a primitive circle shape
  • Material: Used on VisualInstance2D to draw onto the screen
  • Script: Used for extending object behavior from NovaScript

Objects

  • Object
  • Object2D
  • Camera2D
  • VisualInstance2D
  • Sprite2D

Note: All filepaths are relative to the current project path

FileImage

Loads an image asset from a file path.

  • filepath: string — The relative path to the image file (e.g., "Assets/hero.png").

Usage:

[FileImage]
tag = "my_image"
filepath = "sprites/my_image.png"
[/FileImage]

BoxMesh2D

No properties to load, it can only be created and referenced

CircleMesh2D

No properties to load, it can only be created and referenced

Material

Defines the shaders used for rendering.

  • vert_filepath: string Path to a GLSL vertex shader file
  • frag_filepath: string Path to a GLSL fragment shader file

Usage 

[Material]
tag = "glow_material"
vert_filepath = "shaders/vert_glow.glsl"
frag_filepath = "shaders/frag_glow.glsl"
[/Material]

To set uniforms on a shader, create a new script on its visual instance and use this.SetUniform(name, value)

Script

Creates an Interpretor from a NovaScript file

  • filepath: string Path to a NovaScript file

Usage 

[Script]
tag = "rotater_script"
filepath = "Scripts/rotater.ns"
[/Script]

Object

The base parameters available for all nodes.

  • name (string): The identifier for the node.
  • paused (bool): Whether the object receives updates.
  • visible (bool): Whether the object (and children) are drawn.
  • script (Script): Script to extend the object
  • parent (Object): Parent of this object, any objects that dont have parents are added to the scene root

Object2D

Inherits Object values.

  • position (Vector2): Local coordinates.
  • rotation (float): Local rotation in radians.
  • scale (Vector2): Local scale multiplier.

Camera2D

Inherits Object2D values.

  • zoom (float): The magnification level.
  • active (bool): If true, this camera is used for rendering the scene.

VisualInstance2D

Inherits Object2D values.

  • tint (Vector4): Color multiplier for the node and its children.
  • self_tint (Vector4): Color multiplier for only this node.
  • layer (int): The Z-index/render order.
  • material (Material): The assigned material asset.
  • mesh (Mesh2D): The assigned mesh asset.

Sprite2D

Inherits VisualInstance2D values.

  • image (Image): The texture asset to display.
  • offset (Vector2): Rendering displacement.
  • flip_h (bool): Flips the texture horizontally.
  • flip_v (bool): Flips the texture vertically.
