CircleMesh2D

Inherits: Mesh2D > Asset

Description

CircleMesh2D is a primitive geometric asset that generates a smooth circular mesh using a triangle fan originating from the center point.

Automatic Upload: The constructor for this class automatically calls Upload(). Per the Mesh2D specification, this means vertex and index data are moved to the GPU and cleared from system RAM immediately upon instantiation.

Technical Specifications

  • Radius: 1.0 units.
  • Segments: 64 (fixed resolution).
  • Vertex Count: 66 (1 center + 65 perimeter vertices).
  • UV Mapping: Normalized 0.0 to 1.0 across the diameter.

Scene Usage

Define a circle mesh in your scene file to use with sprites or physics shapes.

[CircleMesh2D]
tag = "my_circle"
[/CircleMesh2D]

[Sprite2D]
mesh = "my_circle"
[/Sprite2D]

C++ Usage

// Initialize a unit circle mesh
CircleMesh2D* circle = new CircleMesh2D(); 

// Note: circle->GetVertices() will be empty here 
// because the constructor calls Upload() automatically.
