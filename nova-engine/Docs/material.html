Material

Inherits Asset

Description

A Material is the representation of a Shader and its current Uniforms

using Uniform = std::variant;

Properties

Property Type Description
vert_filepathstd::stringFilepath to be used in the constructer
frag_filepathstd::stringFilepath to be used in the constructer
Reload()voidReloads the vertex and fragment shaders from their files
Bind()voidBinds this material (and any image uniforms) to OpenGL.
Unbind()voidUnbinds this material from OpenGL
SetUniform(std::string name, Uniform)voidSets a shader uniform
SetImageUniform(std::string name, Image* image)voidSets an sampler2D uniform, and increases image's reference count
GetUniform(std::string name)UniformReturns name on this shader

Note: filepaths cannot be accessed or changed after the Material is created, it is only used to create the Material.
