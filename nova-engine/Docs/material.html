Material Class

Inherits: Asset

Description

A Material represents a complete GPU state, combining a Vertex/Fragment shader pair with a specific set of Uniforms (data sent to the shader). It manages the binding of textures and numerical data to the OpenGL pipeline.

// Supported Uniform Types
using Uniform = std::variant[int, float, bool, glm::vec2, glm::vec3, glm::vec4, glm::mat4];

C++ Usage

// Create material with shader paths
Material* mat = new Material("shaders/default.vert", "shaders/default.frag");

// Update shader properties
mat->SetUniform("u_Color", glm::vec4(1.0f, 0.5f, 0.0f, 1.0f));
mat->SetImageUniform("u_MainTexture", myTextureAsset);

// Apply to pipeline
mat->Bind();

Public Methods

Reload()voidRecompiles shaders from disk. Useful for hot-reloading.
Bind()voidActivates the shader program and all associated textures.
Unbind()voidDeactivates the shader program.
