Image

Inherits: Asset | Inherited by: FileImage

Description

The Image class serves as the base for all texture data in the Nova Engine. It manages the transition of raw pixel data from system memory (RAM) to OpenGL texture objects (VRAM).

Workflow Example

// Loading and optimizing an image
FileImage* texture = new FileImage("textures/bricks.png");

if (texture) {
    texture->Upload();   // Send to GPU
    texture->Free();     // Optional: Clear CPU data to save RAM
    texture->Release();  // Standard Asset lifecycle management
}
