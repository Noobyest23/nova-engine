Image

Inherits Asset

Description

Base class for holding image data and managing OpenGL textures

Properties

Property Type Description
GetHeight()intReturns the height of the image in pixels
GetWidth()intReturns the width of the image in pixels
GetChannels()intReturns the channel count of the image, 1 channel is greyscale, 3 is RGB, 4 is RGBA
GetData()unsigned char*Returns the raw pixel data of the image
Free()voidDeletes the pixel data from the image, Does not clear the image from VRAM
Upload()voidUploads the image to the GPU, for use on a material
Bind(uint32_t slot)voidBinds the texture to a GPU texture slot
GetTexture()GLuintReturns this images GPU Texture ID
IsUploaded()boolReturns true if this image has been uploaded to the GPU otherwise false.
