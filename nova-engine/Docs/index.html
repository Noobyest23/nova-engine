What is Nova Engine?

Nova Engine is an in-development 3D game engine. It uses a class-based system for scene objects. As of right now, I would not recommend trying to actually make a game with Nova Engine. It's still very early in development and is mostly just fun to play around with.

The Documentation (just like the engine itself) is also still in development there may be pages missing for some things. Make sure to write a bug report for any pages that are missing

Scene Structure

Scenes are defined using a text-based format. Each object is defined by its class type and properties. Example:

# you can use # for comments
[FileImage]
tag = "image_asset"
filepath = "ProjectFolder/path/to/your/image"
[/FileImage]

[Script]
tag = "Rotater"
filepath = "ProjectFolder/path/to/your/script"
[/Script]

# as a good practice you should put your asset dependencies
# at the top of your scene file

[Sprite2D]
tag = "sprite"
image = "image_asset"
position = Vector2(200, 500)
[/Sprite2D]

[Object2D]
# by default this objects tag is its index in the scene file (3)
script = "Rotater"
parent = "sprite"
[/Object2D]

# any objects that arent added as a child of another object
# go on to the root of the scene

[Camera2D]
[/Camera2D]

# ^ adds a basic camera 2D at 0, 0

Key Notes

  • Each section starts with [ClassName]. And ends with [/Classname]
  • Properties are set with property = value.
  • Assets are loaded from file paths relative to the project folder

For a full list of classes and properties, see the Classes section above.
