Script

Inherits: Asset

Description

The Script asset is used to load and reference NovaScript files. It serves as an interface between the C++ engine and the script interpreter, allowing for dynamic variable manipulation and function execution at runtime.

For details on writing script logic, refer to the NovaScript Documentation.

C++ Usage

// Loading and calling a script function
Script* myScript = new Script("scripts/player_controller.ns");

if (myScript) {
    // Set a global variable in the script
    myScript->Set("playerHealth", Value(100));

    // Call a function within the script
    std::vector args;
    myScript->Call("OnLevelStart", args);
}

Public Methods

Method Return Type Description
Set(std::string name, Value val) void Sets or creates a variable in the script interpreter.
Get(std::string name) Value* Returns a pointer to a script variable, or nullptr if not found.
GetAll() Scope* Returns the complete scope of the interpreter, containing all defined variables.
Call(std::string name, std::vector<Value*> args) void Executes a NovaScript or bound C++ function.

Properties

Property Type Description
filepath std::string The path to the loaded .ns file.
Note: The filepath is utilized only during the construction of the Script asset and cannot be modified once the script is loaded into the interpreter.
