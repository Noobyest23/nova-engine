Script

Inherits Asset

Description

Script assets are used to load and reference NovaScript files

See the NovaScript documentation for more about how to write NovaScript files

Properties

Property Type Description
filepathstd::stringthe filepath to the NovaScript file
Set(std::string name, Value val)voidSets a variable in the novascript interpretor, if the value doesnt exist it gets created.
Get(std::string name)Value*Returns a variable in the NovaScript interpretor if it exists, otherise returns nullptr
GetAll()Scope*Gets the scope of the NovaScript interpretor, which contains every variable and variable name
Call(std::string function_name, std::vector[Value*] arguments)voidCalls a NovaScript function (can be a c++ function, in a module)

Note: filepath cannot be accessed or changed after the image is created, it is only used to create the image.
