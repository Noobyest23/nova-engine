Mesh2D

Inherits Asset

Description

Mesh assets are used to load mesh data. They can be assigned to objects for rendering.

Properties

Property Type Description
GetVertices()std::vector[Vertex2D]&Returns the meshes verticies
GetIndicies()std::vector[unsigned int]&Returns the meshes indicies
Upload()voidUploads the image to the GPU, for use on a material
Bind()voidBinds the mesh for OpenGL
Unbind()voidUnbinds the mesh from OpenGL
GetIndexCount()uint32_tReturns the amount of indicies that the mesh has

Note: GetVertices() and GetIndicies() are unavailable will return empty vectors if the mesh has been uploaded
