Mesh2D

Inherits: Asset | Inherited by: BoxMesh2D, CircleMesh2D

Description

Mesh2D is the base class for all 2D geometry in the Nova Engine. It stores vertex positions, UV coordinates, and index data required for OpenGL rendering.

Memory Management: Like all Assets, Mesh2D objects use reference counting. Ensure you call AddRef() and Release() when handling these in C++.

Public Methods

Method Return Type Description
GetVertices()std::vector<Vertex2D>&Returns a reference to the vertex list.
GetIndices()std::vector<unsigned int>&Returns a reference to the index list.
GetIndexCount()uint32_tReturns the total number of indices.
Upload()voidSends mesh data to the GPU (VRAM).
Bind()voidBinds the Vertex Array Object for rendering.
Unbind()voidUnbinds the current mesh.
Optimization Note: After Upload() is called, the CPU-side vectors for vertices and indices are cleared to save memory. Subsequent calls to GetVertices() or GetIndices() will return empty containers.

Example

// Manual mesh binding example
myMesh->Upload(); 
myMesh->Bind();
glDrawElements(GL_TRIANGLES, myMesh->GetIndexCount(), GL_UNSIGNED_INT, 0);
myMesh->Unbind();
