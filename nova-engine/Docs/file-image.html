Image

Inherits Image > Asset

Description

Creates an image from an image file

#Scene file

[FileImage]
tag = "my_image"
filepath = "path/to/your/image"
[/Fileimage]
[Sprite2D]
image = my_image
[/Sprite2D]

Properties

Property Type Description
filepathstd::stringFilepath to be used in the constructer

Note: filepath cannot be accessed or changed after the image is created, it is only used to create the image.
