FileImage

Inherits: Image > Asset

Description

The FileImage class is a specialized Asset used to load and store image data from local storage. It handles the initial decoding of files (such as PNG or JPG) before they are uploaded to the GPU.

Scene Usage

Define a FileImage in your scene file to associate it with a unique tag for later use in Sprites or Materials.

# Scene file example
[FileImage]
tag = "ui_background"
filepath = "assets/textures/bg_main.png"
[/FileImage]

[Sprite2D]
image = "ui_background"
[/Sprite2D]

C++ Usage

// Constructor signature
FileImage(const std::string& filepath);

// Implementation
FileImage* myImage = new FileImage("assets/textures/hero.png");

Properties

Property Type Description
filepath std::string The relative path to the image resource.
Implementation Note: The filepath is used strictly during construction. Once the image data is loaded into memory, the path cannot be modified. To load a different file, you must instantiate a new FileImage object.
