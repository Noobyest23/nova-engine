Object2D Class

Inherits: Object

Description

Object2D extends the base Object class by adding spatial transformations. It manages a 3x3 transformation matrix, supporting position, rotation, and scaling within a hierarchical scene tree.

Transform Hierarchy

Transformations are inherited from parents. If you move a parent Object2D, all children will move with it, maintaining their local offsets.

Scene Usage

Properties can be set directly in the scene file using Vector2 and numerical values.

[Object2D]
tag = "Player"
position = Vector2(100, 200)
rotation = 0.785 # (45 degrees in radians)
scale = Vector2(2, 2)
[/Object2D]

NovaScript Interface

Object2D provides getters and setters for all transform properties within NovaScript:

func Update(dt) {
    # Rotate the object slowly over time
    local current_rot = this.GetRotation()
    this.SetRotation(current_rot + dt)
    
    # Teleport to world origin
    this.SetGlobalPosition(Vector2(0, 0))
}
Performance Tip: The engine uses a "Dirty Flag" system. Matrices are only recalculated when you call a Set method, making hierarchy deep-traversal very efficient during the Draw call.
